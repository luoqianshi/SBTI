# SBTI - 娱乐人格测试

> MBTI 已经过时，SBTI 来了。

SBTI 是一个纯前端的娱乐性人格测评网站，通过 30+ 道趣味题目，从五个模型（自我、情感、态度、行动驱力、社交）的十五个维度对你进行"审判"，最终为你匹配 27 种独特人格之一。

原作者：B站@蛆肉儿串儿

## 在线体验

[GitHub Pages 在线地址](https://luoqianshi.github.io/SBTI/)

## 功能特色

- **30+ 道趣味题目**：题目风格幽默、接地气，拒绝干巴巴的量表
- **十五维度评分体系**：覆盖自我模型（S1-S3）、情感模型（E1-E3）、态度模型（A1-A3）、行动驱力模型（Ac1-Ac3）、社交模型（So1-So3）
- **27 种人格结果**：从 CTRL（拿捏者）到 DRUNK（酒鬼），每个人格都有独立配图与长文解读
- **隐藏人格机制**：饮酒相关触发题可解锁隐藏人格 DRUNK；匹配度过低时触发 HHHH（傻乐者）兜底人格
- **人格图鉴**：可查看全部 27 种人格的详细介绍
- **纯静态、零依赖**：单 HTML 文件，无需构建工具，直接浏览器打开即可运行

## 本地运行

由于项目是纯静态页面，无需任何构建步骤：

1. 克隆仓库
   ```bash
   git clone https://github.com/your-username/SBTI.git
   cd SBTI
   ```

2. 用浏览器直接打开 `index.html`，或使用本地服务器：
   ```bash
   # Python
   python -m http.server 8080

   # Node.js (npx)
   npx serve .
   ```

3. 访问 `http://localhost:8080`

## 部署到 GitHub Pages

### 方式一：通过 GitHub 网页设置

1. 将代码推送到 GitHub 仓库
2. 进入仓库 **Settings** → **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 `main`，文件夹选 `/ (root)`
5. 点击 **Save**，等待部署完成

### 方式二：通过 GitHub Actions 自动部署

项目已包含 GitHub Actions 工作流配置（`.github/workflows/deploy.yml`），推送到 `main` 分支后会自动部署。

## 十五维度说明

| 模型 | 维度 | 说明 |
|------|------|------|
| 自我模型 | S1 自尊自信 | 对自我价值的整体评估 |
| | S2 自我清晰度 | 对自身特质与需求的认知程度 |
| | S3 核心价值 | 内驱力与目标感的强弱 |
| 情感模型 | E1 依恋安全感 | 在亲密关系中的安全感水平 |
| | E2 情感投入度 | 恋爱/亲密关系中的投入倾向 |
| | E3 边界与依赖 | 对个人空间与独立性的需求 |
| 态度模型 | A1 世界观倾向 | 对外部世界和人性的基本信任程度 |
| | A2 规则与灵活度 | 对规则和秩序的态度 |
| | A3 人生意义感 | 对生活目标和意义感的感知 |
| 行动驱力模型 | Ac1 动机导向 | 趋近成功 vs 回避风险的倾向 |
| | Ac2 决策风格 | 决策速度和果断程度 |
| | Ac3 执行模式 | 计划性与执行力的强弱 |
| 社交模型 | So1 社交主动性 | 主动发起社交的意愿 |
| | So2 人际边界感 | 在关系中保持距离 vs 渴望亲近 |
| | So3 表达与真实度 | 在不同社交场景中自我表达的一致性 |

## 技术实现

- 纯 HTML + CSS + JavaScript，零框架依赖
- CSS 变量驱动的设计系统，支持响应式布局
- 题目随机排序，条件分支题（饮酒触发题）动态插入
- 基于维度向量距离的最近邻匹配算法
- 所有数据内嵌在 HTML 中，无外部 API 调用
